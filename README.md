# Homebound

> 보호소 동물 79만 건의 데이터를 분석하여 입양률 향상 요인을 식별하고, 지역 간 성과 격차를 정량화하여 데이터 기반 정책 제언을 도출하는 프로젝트

## 1. 프로젝트 개요

- **주제**: 전국 보호종료동물 데이터 기반 입양률 향상 및 보호소 운영 효율화 전략 도출
- **배경 및 현황**: 대한민국에서 매년 약 10만 마리 이상의 유실/유기동물이 발생하며, 보호소에 들어온 동물의 52.1%가 자연사(29.3%) 또는 안락사(22.8%)로 생을 마감한다. 입양률은 31.2%에 불과하고, 지역·보호소 간 성과 격차가 극심하다(입양률 5.7%~72.1%).
- **목표**: 입양 성공에 영향을 미치는 요인(동물 특성, 보호소/지역 역량, 시간 패턴)을 데이터로 식별하고, 광역 공동 보호체계·봄철 TNR·고령동물 입양촉진 등 구체적 정책 방향에 정량적 근거를 제공한다.

## 2. 문제 정의

- **비즈니스 리스크**: 입양률 정체(2024년 27.7%), 사망률 53.8% 지속, 지역 간 입양률 격차 24%p(시도), 61%p(시군구). 정부의 2029년 유기동물 6만 마리 감축 목표 달성이 불확실하다.
- **핵심 질문**:
  - RQ1: 어떤 특성(나이, 품종, 중성화 등)의 동물이 입양될 확률이 높고, 어떤 동물이 사각지대에 놓이는가?
  - RQ2: 같은 조건의 동물이라도 보호소·지역에 따라 입양률이 크게 다른 원인은 무엇인가?
  - RQ3: 유기동물 발생과 입양에 시간적 패턴이 존재하며, 선제 대응이 가능한가?

## 3. 통합 솔루션

2019~2025년 전국 보호종료동물 789,874건 데이터를 기반으로, **3축(동물 특성 × 지역/보호소 역량 × 시간 패턴)** 관점에서 탐색적 분석을 수행한다. 외부 보고서의 통계를 교차 검증하고, 7개 분석 가설을 통계적으로 검정하여 채택/기각한다. 피처 엔지니어링(색상 표준화, NLP 키워드 추출, 지오코딩)으로 분석 변수를 확장하고, 최종적으로 태블로 대시보드와 정책 제안 리포트를 산출한다.

## 5. 핵심 분석 방법론

### Phase 1: 데이터 전처리 (`01_raw_data → 02_outputs/data`)
- 원본 xlsx 파일(확장자 .csv)의 형식 이슈 해결, 결측치/이상치 처리
- 나이(출생연도→추정나이), 체중(극단값 NaN 처리), 종료사유(파싱) 등 19개 원본 → 22개 분석 컬럼 변환
- 6종 집계 CSV 생성 (월별, 다차원, 보호소별, 지역별, 품종별, 시간패턴별)

### Phase 2: 탐색적 데이터 분석 (EDA)
- **보고서 교차 검증**: 외부 보고서의 7개 핵심 주장을 데이터로 독립 검증 → 6개 일치, 1개(사망률)는 보고서보다 심각
- **종합 EDA**: 전국/시도/시간/보호소 차원의 종합 탐색
- **가설 검증 EDA**: 7개 가설에 대해 카이제곱, Mann-Whitney U, 스피어만 상관, 선형회귀 등 통계적 검정 수행

### Phase 3: 피처 엔지니어링
- **보호기간일 이상치 처리**: [0, 180] 범위 클리핑 (음수 13건, 365일 초과 20건 처리)
- **색상 표준화**: 82개 원본 색상 → 12개 표준 범주 (사전 매핑 + 규칙 기반 함수)
- **특징 텍스트 NLP**: konlpy Okt 형태소 분석으로 명사/형용사 추출, WordCloud, 키워드 플래그 7종 생성
- **발견장소 지오코딩**: 카카오 API로 232개 시군구 좌표 획득, folium 인터랙티브 지도 생성

## 6. 분석 결과 및 기대 효과

### 가설 검증 결과 (7개 중 5개 채택)

| 가설 | 결과 | 핵심 수치 |
|------|------|----------|
| H1: 어린 동물 → 입양률↑ | **채택** | 1세이하 33.7% → 11세+ 10.2% |
| H2: 중성화 → 입양률↑ | **수정채택** | 통계적 유의, 실질 차이 0.7%p |
| H3: 광역시 > 도 | **기각** | 도(31.2%) > 광역시(29.4%) |
| H4: 여름 고양이 발생↑ | **채택** | 고양이 +48.5%, 개 +5.0% |
| H5: 규모↑ → 입양률↑ | **기각** | 방향 반대 (rho = -0.165) |
| H6: 도 내 격차 극심 | **채택** | 평균 CV 0.408, 격차 46.8%p |
| H7: 연도별 감소 추세 | **채택** | -5,220건/년, R²=0.952 |

### 5대 핵심 인사이트

1. **나이가 입양의 가장 강력한 예측 변수** — 고령 동물 27,577건 중 대부분이 보호소에서 생을 마감
2. **"도시 vs 농촌" 프레임은 틀렸다** — 전북(43.3%)이 서울(30.9%)보다 높음. 보호소 운영 역량이 결정적
3. **같은 도 안에서 입양률 6배 차이** — 경북 칠곡(72.1%) vs 청송(11.0%). 광역 공동 보호체계 필요
4. **고양이는 여름에 집중 발생** — 봄철 TNR 사업의 타이밍 근거
5. **"큰 보호소 = 좋은 보호소"가 아님** — 적정 규모 분산형 + 네트워크 모델이 효과적

### 피처 엔지니어링 결과

- 색상 82개 → 12개 표준 범주 통합 (흰색 계열 입양률 35.8% 최고)
- NLP 키워드 분석: "온순" 키워드 포함 동물의 입양률이 현저히 높음
- 232개 시군구 지오코딩 완료 → 지도 기반 지역별 입양률 시각화
