# 태블로 대시보드 설계 가이드

## 데이터 연결 방법

### 메인 데이터소스
`shelter_animals_tableau.csv`를 태블로에 드래그하여 연결. 이 파일 하나로 대부분의 시각화가 가능하다.

### 보조 데이터소스 (필요시)
집계 CSV 파일들은 태블로 성능 최적화가 필요할 때 사용. 79만건 원본 대비 수백~수천 행으로 빠른 렌더링 가능.

### 데이터 타입 설정
태블로 연결 후 다음 컬럼의 타입을 확인할 것:
- `접수일자`: 날짜(Date) 타입
- `입양여부`: 숫자(Number) → 측정값으로 활용
- `추정나이`, `체중_정제`, `보호기간일`: 연속형 숫자
- `접수년도`, `접수월`, `접수분기`: 불연속형(Discrete)으로 설정 권장

## 대시보드 1: 종합 개요 (Overview)

### KPI 카드 (상단)
| 지표 | 계산식 | 설명 |
|------|--------|------|
| 총 보호 건수 | COUNT([순번]) 또는 행 수 | 전체 데이터 건수 |
| 입양률 | SUM([입양여부]) / COUNT(*) | 전체 입양 성공 비율 |
| 안락사율 | 종료사유_정제 = '안락사' 비율 | 안락사 비율 |
| 연평균 유기동물 수 | 총건수 / 연도 수 | 연간 평균 발생 건수 |

### 차트 구성
1. **연도별 유기동물 추이** (라인차트): X=접수년도, Y=건수, 색상=동물종류
2. **종료사유 비율** (도넛/파이): 종료사유_정제별 비율
3. **시도별 건수 지도** (맵): 시도 지리 역할 → 색상에 건수 또는 입양률
4. **동물 종류 비율** (가로 막대): 개/고양이/기타축종/기타

## 대시보드 2: 입양률 분석 (Adoption Analysis)

### 핵심 질문
- 어떤 조건의 동물이 입양될 확률이 높은가?
- 중성화 여부가 입양률에 미치는 영향은?
- 나이와 입양률의 관계는?

### 차트 구성
1. **나이그룹별 입양률** (막대차트)
   - X=나이그룹, Y=AVG(입양여부)*100, 색상=동물종류
   - 정렬: 60일미만 → 1세 이하 → 2-3세 → ... → 11세 이상

2. **중성화 여부별 입양률** (막대차트)
   - X=중성화 여부, Y=AVG(입양여부)*100
   - 필터: 동물종류

3. **성별 × 동물종류 입양률** (히트맵)
   - 행=성별, 열=동물종류, 색상=AVG(입양여부)

4. **체중그룹별 입양률** (막대차트)
   - X=체중그룹, Y=AVG(입양여부)*100
   - 동물종류별 분리 또는 필터

5. **품종별 입양률 TOP/BOTTOM 20** (수평 막대)
   - `agg_breed_adoption.csv` 활용 또는 원본에서 계산
   - 총건수 100건 이상 필터 적용 권장 (소수 표본 제거)

### 계산 필드 (태블로)
```
// 입양률 (%)
SUM([입양여부]) / COUNT([입양여부]) * 100

// 안락사율 (%)
SUM(IF [종료사유_정제] = '안락사' THEN 1 ELSE 0 END) / COUNT(*) * 100
```

## 대시보드 3: 보호소 운영 분석 (Shelter Performance)

### 핵심 질문
- 입양률이 높은 보호소의 특징은?
- 지역별 안락사율 격차는 얼마나 되는가?
- 보호소 규모와 입양률의 관계는?

### 차트 구성
1. **보호소별 입양률 vs 안락사율** (산점도)
   - `agg_shelter_performance.csv` 활용
   - X=입양률, Y=안락사율, 크기=총보호건수, 색상=시도
   - 총보호건수 100건 이상 필터

2. **시도별 입양률/안락사율 비교** (이중 막대)
   - X=시도, Y=입양률/안락사율
   - 정렬: 입양률 기준 내림차순

3. **시도별 연도별 입양률 추이** (라인차트)
   - `agg_region_yearly.csv` 활용
   - X=접수년도, Y=입양률, 색상=시도 (상위 5개 시도 하이라이트)

4. **보호소 TOP 20** (테이블)
   - 총보호건수, 입양률, 안락사율, 자연사율 표시
   - 조건부 서식: 입양률 높으면 녹색, 안락사율 높으면 적색

## 대시보드 4: 시계열 패턴 (Temporal Patterns)

### 핵심 질문
- 유기동물 발생에 계절성이 있는가?
- 요일별 패턴이 존재하는가?
- 개와 고양이의 시간 패턴이 다른가?

### 차트 구성
1. **월별 유기동물 발생 패턴** (라인차트)
   - X=접수월, Y=AVG(건수), 색상=동물종류
   - 연도별 평균으로 계절성 확인

2. **요일별 발생 패턴** (막대차트)
   - X=접수요일, Y=AVG(건수)
   - 요일 정렬: Monday → Sunday

3. **연도-월 히트맵** (하이라이트 테이블)
   - 행=접수년도, 열=접수월, 색상=건수
   - 계절성과 연도 추세를 동시 파악

4. **종료사유별 월간 추이** (면적차트)
   - `agg_monthly_outcome.csv` 활용
   - X=년-월, Y=건수, 색상=종료사유_정제
   - 100% 누적 면적 차트로 비율 변화 확인

## 필터 설계

### 전역 필터 (모든 대시보드 공통)
- **접수년도**: 슬라이더 또는 드롭다운
- **동물종류**: 개 / 고양이 / 기타축종 / 기타
- **시도**: 멀티셀렉트 드롭다운

### 대시보드별 추가 필터
- 입양률 분석: 성별, 중성화 여부, 나이그룹
- 보호소 분석: 최소 보호건수 (슬라이더)
- 시계열: 동물종류, 종료사유

## 인사이트 도출 포인트

### 가설 1: 중성화된 동물의 입양률이 더 높을 것이다
- 검증 방법: 중성화 여부별 입양률 비교 (동물종류 통제)
- 시각화: 그룹 막대차트

### 가설 2: 어린 동물일수록 입양률이 높을 것이다
- 검증 방법: 나이그룹별 입양률 추이
- 시각화: 막대차트 + 추세선

### 가설 3: 여름철(6-8월) 유기동물 발생이 증가할 것이다
- 검증 방법: 월별 발생 건수 비교
- 시각화: 라인차트 (개/고양이 분리)

### 가설 4: 도시 지역이 농촌 지역보다 입양률이 높을 것이다
- 검증 방법: 시도별 입양률 비교 (광역시 vs 도)
- 시각화: 지도 + 막대차트

### 가설 5: 보호소 규모(총 보호건수)와 입양률 간 관계가 있을 것이다
- 검증 방법: 산점도 (보호건수 vs 입양률)
- 시각화: 버블차트
