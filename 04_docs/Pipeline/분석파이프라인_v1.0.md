## 3. 피처 엔지니어링 (Feature Engineering)

**노트북**: `03_notebooks/04_피처엔지니어링.ipynb`

### Phase 1 피처 (전처리 단계에서 생성)

| 파생변수명 | 정의 | 생성 로직 |
|-----------|------|----------|
| 추정나이 | 동물의 추정 나이(세) | 접수년도 - 출생연도 |
| 60일미만 | 60일 미만 어린 동물 플래그 | 나이 컬럼 '60일미만' 텍스트 감지 |
| 나이그룹 | 7개 연령 구간 | 60일미만/1세이하/2-3세/4-5세/6-7세/8-10세/11세+ |
| 체중_정제 | 이상치 제거 체중(kg) | 0 이하/200 초과 → NaN |
| 체중그룹 | 6개 체중 구간 | 2kg이하/2-5/5-10/10-20/20-30/30kg+ |
| 종료사유_정제 | 간소화된 종료사유 | '종료(' 접두사 + ')' 접미사 제거 |

| 동물종류 | 개/고양이/기타축종/기타 | 품종명 키워드 매칭 (우선순위: 고양이→개→기타축종→기타) |

## 5. 모델링 및 데이터 분석 (Modeling & Analysis)

### 분석 모델링

현재 단계에서는 기술통계 + 통계적 검정 기반의 탐색적 분석에 집중. 적용한 분석 기법:

| 기법 | 용도 | 적용 대상 |
|------|------|----------|
| 카이제곱 독립성 검정 | 범주형 변수 간 관련성 검정 | H1(나이), H2(중성화), H4(계절) |
| Mann-Whitney U 검정 | 두 그룹 분포 비교 (비모수) | H3(광역시 vs 도) |
| 스피어만 상관분석 | 연속형 변수 간 단조 관계 측정 | H5(규모 vs 입양률) |
| 선형회귀 | 시계열 추세 분석 | H7(연도별 감소) |
| 비율 Z-검정 | 두 비율 간 차이 검정 | H2(중성화 비율 차이) |
| 변동계수(CV) | 그룹 내 분산 정량화 | H6(도 내 시군구 격차) |

### 가설 검정 결과 요약

| 가설 | 판정 | p-value | 핵심 수치 |

## 6. 인사이트 도출 및 시각화 (Insights & Visualization)

### 핵심 결과

1. **나이가 입양의 최강 예측 변수**: 1세 이하(33.7%) → 11세 이상(10.2%). 고령 동물 27,577건 중 대부분 보호소에서 사망. 고령·대형견에 대한 별도 입양 촉진 프로그램 필요
2. **"도시=높은 입양률" 편견 깨짐**: 전북(43.3%)이 서울(30.9%)보다 높음. 보호소 운영 역량이 도시/농촌 구분보다 결정적
3. **도 내 시군구 격차 극심**: 경북 칠곡(72.1%) vs 청송(11.0%), 8개 도 모두 30%p+ 격차. 광역 공동 보호체계의 정량적 근거
4. **고양이 여름 집중 발생**: 월평균 대비 +48.5%. 봄철(3~5월) TNR 집중 사업의 타이밍 근거
5. **대형 보호소의 역설**: 규모↑ → 입양률↓ (rho = -0.165). 적정 규모 분산형 모델이 효과적
6. **보호소 사망률 53.8%**: 보고서(46%)보다 심각. 자연사율(30.1%)이 안락사율(23.7%)보다 높아 보호소 내 의료·위생 관리 부실 시사
7. **믹스견 구조적 소외**: 전체 유기견의 74.5%가 믹스견이지만 입양률은 27.7%(품종견 36.4% 대비 -8.8%p)
