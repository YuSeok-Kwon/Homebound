# Homebound 프로젝트 — Executive Summary

> **작성일:** 2026-03-24
> **대상:** 정책결정자, 보호소 관계자, 동물보호 활동가
> **한 줄 요약:** 보호소 입양률은 "어디에 있느냐"가 아니라 **"어떻게 운영하느냐"**에 달려 있으며, 보호소 유형별 맞춤 전략이 필요합니다.

---

## 1. 배경: 왜 이 분석을 했는가

전국 **564개 보호소**, **789,874건**의 보호동물 데이터를 분석했습니다. 전국 평균 입양률은 약 36%이지만, 보호소별로 5%~90%까지 극심한 격차가 존재합니다. 이 격차의 원인을 파악하고, 보호소 유형별 맞춤 정책을 제시하기 위해 분석을 수행했습니다.

---

## 2. 핵심 질문과 답변

### Q1. 입양률 격차의 원인은 무엇인가?

> **"지역이 아니라 운영 방식이 핵심입니다"**

지역(시도) 변수만으로는 입양률 분산의 **3.1%**만 설명됩니다. 반면, 보호소의 운영 변수(안락사 관행, 의료 인프라, 보호기간 등) 19개를 활용하면 **56.4%**를 설명할 수 있습니다. 같은 지역 내에서도 보호소 간 입양률 격차가 크며, 이는 운영 방식의 차이에서 비롯됩니다.

### Q2. 모든 보호소에 동일한 전략이 유효한가?

> **"아닙니다. 4가지 유형별로 전혀 다른 전략이 필요합니다"**

564개 보호소를 운영·성과 특성 8개로 분석한 결과, **4개의 뚜렷한 유형**이 확인되었습니다:

```
┌─────────────────────────────────────────────────────────────────┐
│                      564개 보호소 유형 분포                       │
├─────────────────────────────────────────────────────────────────┤
│                                                                 │
│  ■■■■■■■■■■■■■■■■■■■■■■■■■■■■■■  C0: 고안락사형 319개 (56.6%)  │
│  입양률 26% │ 안락사율 25% │ 수의사 49%                          │
│                                                                 │
│  ■■■■■■■■■■■■■■■  C3: 고입양형 159개 (28.2%)                    │
│  입양률 57% │ 안락사율 4% │ 소규모                               │
│                                                                 │
│  ■■■■■  C1: 균형관리형 60개 (10.6%)                              │
│  입양률 41% │ 소규모 │ 수의사 5%                                 │
│                                                                 │
│  ■■  C2: 균형관리형 26개 (4.6%)                                  │
│  입양률 38% │ 수의사 100%                                        │
│                                                                 │
└─────────────────────────────────────────────────────────────────┘
```

각 유형별 입양률의 **핵심 결정요인**이 다릅니다:

| 보호소 유형 | 핵심 결정요인 | 전체 모델과 다른 점 |
|---|---|---|
| C0: 고안락사형 (319개) | 안락사 관행, **수의사 배치**, 보호소 규모 | 수의사 중요도 전체 16위 → C0 7위 |
| C1: 균형관리형 (60개) | **1인가구비율, 고령화율** | 사회경제 환경이 핵심 |
| C3: 고입양형 (159개) | **반환률**, 인구밀도, 보호기간 | 입양 후 반환 관리가 1순위 |
| C2: 균형관리형 (26개) | (소표본, 추가 데이터 필요) | — |

### Q3. 가장 시급한 정책은 무엇인가?

> **"전체 보호소의 57%를 차지하는 고안락사형(C0)의 의료 인프라 확충이 최우선"**

우선순위 점수(= 보호소 수 × 개선 여지)로 평가하면:

```
우선순위 1 ━━━━━━━━━━━━━━━━━━━━━━━━  C0: 고안락사형 (점수 235.7)
우선순위 2 ━━━━━━━━                    C3: 고입양형   (점수 67.8)
우선순위 3 ━━━━                        C1: 균형관리형 (점수 35.6)
우선순위 4 ━━                          C2: 균형관리형 (점수 16.2)
```

## 4. 정책 우선순위 매트릭스

### C0: 고안락사형 (319개, 입양률 26%) — 최우선 대상

| 정책 | 근거 | 기대 효과 |
|------|------|-----------|
| 안락사 대안 프로토콜 도입 | 안락사율이 SHAP 1위 결정요인 | 안락사 감소 → 입양 기회 확대 |
| **수의사 상주 배치** | 수의사유무가 숨은 핵심 변수 | 동물 건강 → 입양 매력도 향상 |
| 보호소 규모별 자원 배분 | 대규모 보호소 과부하 | 인력·공간 적정화 |
| TNR + 협력 입양 네트워크 | 자연사율 감소 필요 | 유입량 관리 + 입양 채널 다변화 |

### C3: 고입양형 (159개, 입양률 57%) — 성과 유지·확대

| 정책 | 근거 | 기대 효과 |
|------|------|-----------|
| 입양 후 관리 프로그램 | 반환률이 SHAP 1위 | 반환 방지 → 순입양률 상승 |
| 보호기간 최적화 | 평균보호기간이 SHAP 4위 | 체류 시간 단축 → 회전율 향상 |
| 고령 지역 특화 프로그램 | 고령화율이 SHAP 5위 | 고령층 맞춤 반려동물 매칭 |

### C1: 균형관리형 (60개, 입양률 41%) — 수요 측면 개선

| 정책 | 근거 | 기대 효과 |
|------|------|-----------|
| 1인 가구 입양 지원 | 1인가구비율이 SHAP 2위 | 잠재 수요 활성화 |
| 고령 지역 반려동물 복지 연계 | 고령화율이 SHAP 3위 | 독거 고령층 + 반려동물 매칭 |
| 보호소 정보 투명성 제고 | 인프라정보유무가 SHAP 6위 | 신뢰도 향상 → 입양 전환율 상승 |

## 5. 분석 방법론 (간략)

1. **데이터 통합**: 보호동물 데이터 + 보호소 인프라 + 사회경제 지표를 보호소 단위로 통합
2. **보호소 유형화**: K-Means 클러스터링 (8개 운영·성과 변수)으로 4개 유형 도출
3. **결정요인 분석**: LightGBM 회귀 + SHAP으로 유형별 입양률 핵심 요인 식별
4. **검증**: 5-Fold 교차검증, Track A/B 비교, v1→v2 ARI 비교
