## 2. 보호소 클러스터링

### 2.1 방법론

- **피처:** 입양률, 안락사율, 자연사율, 평균보호기간 (4개 축)
- **전처리:** StandardScaler 정규화, 평균보호기간 99퍼센타일(15.6일) 클리핑
- **알고리즘:** K-Means (K=2~10 탐색)
- **최적 K 결정:** Silhouette Score + Calinski-Harabasz Index + Elbow Method 병행

### 2.2 최적 K 선택

| K | Silhouette | CH Index | 비고 |
|---|-----------|----------|------|
| 2 | 0.2618 | 190.4 | |
| 3 | 0.2899 | 211.7 | |
| **4** | **0.3130** | **205.0** | **선택 (해석력 + 지표 균형)** |
| 5 | 0.3033 | 222.4 | |
| 6 | 0.3203 | 225.5 | |

- Silhouette Score 0.3130 > 기준치 0.25 충족
- K=4로 적극입양형 / 고안락사형 / 고자연사형 / 균형관리형 — 직관적 해석 가능

### 2.3 클러스터 프로파일

| 클러스터 | 보호소 수 (비율) | 입양률 | 안락사율 | 자연사율 | 총보호건수 평균 |
|----------|-----------------|--------|---------|---------|---------------|
| **C0: 균형관리형** | 53개 (9.4%) | 43.8% | 18.3% | 11.3% | - |
| **C1: 적극입양형** | 177개 (31.4%) | 61.4% | 6.0% | 11.7% | 854 |
| **C2: 고안락사형** | 113개 (20.0%) | 20.1% | 53.3% | 17.1% | 1,587 |
| **C3: 고자연사형** | 221개 (39.2%) | 24.4% | 11.8% | 40.0% | 1,831 |

### 2.4 핵심 발견

1. **적극입양형(C1)이 전체의 31.4%** — 입양률 61.4%로 평균(37%)의 1.7배. 상대적으로 소규모 보호소가 많음 (평균 854건)
2. **고안락사형(C2)은 대형 보호소에 집중** — 평균 보호건수 1,587건. 제주 동물보호센터(안락사율 56.3%) 등 포함
3. **고자연사형(C3)이 가장 다수(39.2%)** — 의료 인프라 부족이 의심됨. 보호건수가 가장 많은 그룹
4. **같은 시도 내에서도 클러스터가 혼재** — "Where가 아닌 How" 가설을 정량적으로 지지

### 2.5 PCA 분석

- PC1(37.9%): 입양률(+) vs 안락사율(-) 축 → "입양 지향성"
- PC2(29.6%): 자연사율(+) vs 안락사율(-) 축 → "사망 유형"
- 누적 설명 분산: 67.5%

### 2.6 클러스터별 정책 제언

| 클러스터 | 정책 방향 | 구체적 제언 |
|----------|----------|------------|
| C0: 균형관리형 | 입양률 추가 제고 | 입양 홍보 강화, 절차 간소화, 온라인 입양 매칭 |
| C1: 적극입양형 | 벤치마크 확산 | 운영 노하우 공유 프로그램, 모범 사례 매뉴얼 |
| C2: 고안락사형 | 안락사 대안 마련 | TNR 확대, 협력 입양 네트워크, 안락사 의사결정 프로토콜 |
| C3: 고자연사형 | 의료 환경 개선 | 수의사 상주 지원, 의료비 보조, 보호 환경 표준화 |

## 3. 입양 확률 예측 모델

### 3.1 방법론

- **타겟:** 입양여부 (0/1), 불균형 비율 31.2% vs 68.8%
- **분할:** 시간 기반 — Train(2019-2023, 581,217행) / Test(2024, 100,793행)
- **피처:** 47개 (수치 5 + 이진 11 + 파생 2 + Target Encoding 3 + One-Hot 26)
- **불균형 처리:** `scale_pos_weight = 2.17` (class_weight='balanced')
- **튜닝:** RandomizedSearchCV (30회 × 5-Fold)

### 3.2 피처 구성

| 구분 | 피처 | 개수 |
|------|------|------|
| 수치형 | 추정나이, 체중_정제, 보호기간일_클리핑, 접수년도, 접수월 | 5 |
| 이진 | 60일미만, is_온순~is_질병 (7), 고령동물, 소형동물, 여름유기 | 11 |
| 파생 수치 | 긍정성격점수, 부정성격점수 | 2 |
| Target Encoding | 시도, 품종그룹, 색상_표준 | 3 |
| One-Hot | 동물종류, 성별, 중성화 여부, 나이그룹, 체중그룹, 접수분기, 접수요일, 지역유형 | 26 |

### 3.4 최종 모델 성능 (LightGBM Tuned)

| 지표 | 값 | 목표 | 달성 |
|------|-----|------|------|
| **ROC-AUC** | **0.7070** | >= 0.70 | ✓ |
| **Recall** | **0.5573** | >= 0.60 | △ (근접) |
| **F1** | **0.4912** | - | - |

- ROC-AUC 목표 달성 (0.7070)
- Recall은 0.5573으로 목표(0.60)에 근접하나 미달
  - 임계값(threshold)을 0.5 → 0.4로 조정 시 Recall 향상 가능 (Precision 트레이드오프)
  - 현실적 활용 시 상위 위험군 스크리닝 용도로 충분

## 4. 종합 결론: "Where가 아닌 How"

### 4.1 클러스터링이 밝힌 것

같은 지역 내에서도 보호소 유형이 혼재한다. 적극입양형(입양률 61%)과 고안락사형(안락사율 53%)이 동일 시도에 공존하며, 이는 **지역이 아닌 운영 방식**이 동물의 운명을 결정함을 의미한다.

### 4.2 예측 모델이 밝힌 것

입양 여부를 결정하는 가장 큰 요인은 **품종(편향)**, **체중(크기 선호)**, **지역(인프라 차이)** 순이다. 이는 보호소가 통제 가능한 변수(홍보, 매칭, 입양 절차)를 통해 입양률을 개선할 수 있는 여지가 크다는 것을 시사한다.

### 4.3 두 분석의 교차점

| 클러스터 유형 | 대응하는 SHAP 인사이트 | 통합 제언 |
|-------------|---------------------|----------|
| 적극입양형 | 품종 편향이 낮은 입양 문화 | → 벤치마크 사례로 확산 |
| 고안락사형 | 대형/고령 동물 집중 | → TNR + 특별 입양 프로그램 |
| 고자연사형 | 의료 인프라 부족 | → 수의사 지원 + 환경 개선 |
| 균형관리형 | 입양 홍보 부족 | → 온라인 매칭 + 인식 개선 |
