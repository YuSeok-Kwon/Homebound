## 1. 분석 배경 및 목적

### 1.1 선행 분석 요약

가설 검증 RQ 리포트(v1)에서 7개 가설 중 2개가 기각되었으며, 기각된 가설이 가장 강력한 인사이트를 제공했다.

| 가설 | 판정 | 핵심 결과 |
|------|------|----------|
| H3: 광역시 > 도 입양률 | 기각 | 오히려 도(31.2%) > 광역시(29.4%), p = 0.364 |
| H5: 보호소 규모 → 입양률 | 기각 | 방향 반대, ρ = -0.165 |
| H6: 도 내 시군구 격차 | 채택 | 평균 CV 0.408, 격차 46.8%p |

이 결과들은 공통적으로 **"입양률은 도시/농촌 구분이나 규모보다 개별 보호소·지자체의 운영 역량에 좌우된다"**는 해석을 제시했다. 본 분석은 이 해석을 외부 사회경제 데이터로 재검증한다.

### 1.2 연구 질문

1. 재정자립도가 낮은 시군구일수록 입양률이 낮은가? (H6의 구조적 원인 검증)
2. 인구밀도가 H3(광역시 vs 도) 기각 결과를 설명하는가?
3. 1인가구비율 증가가 유기동물 발생과 관련 있는가?
4. 고령화율이 높을수록 입양 수요가 낮은가?
5. 다중회귀를 통해 입양률에 가장 큰 영향을 미치는 사회경제 변수는?

### 2.2 분석 설계

전략 2(횡단면 분석)를 채택했다. 2023년 단일 연도의 사회경제 데이터를 보호동물 데이터(2019~2024 전체 기간 집계)와 시군구 단위로 병합하는 방식이다.

**횡단면 선택 이유**:

- 현재 보호동물 성과 데이터(`agg_shelter_performance.csv`)가 연도 구분 없이 전체 기간 합산이므로 바로 병합 가능
- 인구밀도, 재정자립도 같은 구조적 변수는 단기간(6년)에 급변하지 않아 1개년 스냅샷으로도 시군구 간 상대적 위치를 충분히 반영
- 2023년은 분석 대상 기간(2019~2024)의 중간 시점에 해당하여 대표성이 높음

### 2.3 병합 파이프라인

```
[보호동물 데이터] ─── agg_shelter_performance.csv (564개 보호소)
    │  시군구별 합산 → 232개 시군구
    ▼
[행정구역 명칭 정규화]
    │  시도명 통일: 강원도 → 강원특별자치도, 전라북도 → 전북특별자치도
    │  구 단위 통합: 수원시 장안구/권선구/... → 수원시
    │  광역시 구: 그대로 유지 (시도+시군구 복합 키)
    ▼
[외부 데이터 병합] ─── LEFT JOIN on (시도, 시군구)
    │  인구 → 인구밀도 산출 (총인구/면적)
    │  재정자립도 직접 병합
    │  고령화율/1인가구비율: 시도 단위 병합
    ▼
[최종 데이터셋] ─── 232행 × 18열
    merged_sigungu_socioeconomic.csv
```

### 4.3 상관 분석 해석

**유의한 변수 (2개)**:

- **재정자립도 (ρ = +0.192, p = 0.006)**: 재정이 건전한 시군구일수록 입양률이 미세하게 높다. 그러나 상관계수가 0.2 미만으로 약한 수준이며, 입양률 분산의 약 3.7%만 설명한다. 재정자립도는 보호소 인력·시설 투자 여력과 관련될 수 있으나, 직접적 인과가 아닌 간접적 배경 변수로 해석하는 것이 적절하다.

- **총보호건수 (ρ = +0.206, p = 0.004)**: H5(규모 vs 입양률)에서 보호소 단위로는 음의 상관(ρ = -0.165)이었으나, 시군구 단위 합산에서는 약한 양의 상관으로 전환되었다. 이는 집계 수준(보호소 vs 시군구)에 따라 상관 방향이 바뀌는 **생태학적 오류(ecological fallacy)** 가능성을 시사한다.

**무의미한 변수 (3개)**:

- **인구밀도 (ρ ≈ 0, p = 0.910)**: 입양률과 완전 무관. H3(광역시 vs 도 차이 없음)을 재확인하는 결과. 인구가 밀집되어 잠재 입양자가 많아도 입양률이 높아지지 않는다.

- **고령화율 (ρ = +0.027, p = 0.721)**: 고령 지역이라고 입양률이 낮지 않다. 다만 이 변수는 시도 단위로만 측정되어 시군구 내 편차를 반영하지 못하는 한계가 있다.

- **1인가구비율 (ρ = +0.044, p = 0.552)**: 1인가구 증가가 유기 발생이나 입양률에 미치는 영향은 본 데이터에서 확인되지 않았다. 이 역시 시도 단위 측정의 한계.

## 6. 핵심 발견 및 해석

### 6.1 3대 핵심 발견

**발견 1: 사회경제 조건은 입양률을 거의 설명하지 못한다 (R² = 0.031)**

인구밀도, 재정자립도, 보호건수를 모두 투입한 다중회귀의 R²가 0.031에 불과하다. 같은 재정자립도, 같은 인구밀도의 시군구에서도 입양률이 10%에서 70%까지 극심한 차이를 보이며, 이 차이는 사회경제 조건으로 설명되지 않는다.

이 결과는 RQ 리포트의 H3(기각), H5(기각), H6(채택) 결과와 정합적이다. 입양률은 "어디에 있는가(where)"보다 **"어떻게 운영하는가(how)"**에 의해 결정된다.

**발견 2: 인구밀도는 입양률과 완전 무관 (ρ ≈ 0)**

인구밀도와 입양률의 스피어만 상관계수가 -0.008(p = 0.910)로 사실상 제로다. 이는 H3 기각("도시 = 높은 입양률"은 편견)에 대한 가장 직접적인 외부 검증 결과다. 인구가 2만 명인 농촌 시군구와 200만 명인 대도시의 입양률이 통계적으로 동일하다는 것은, 잠재 입양자 풀의 크기가 실제 입양으로 전환되는 비율과 무관함을 의미한다.

**발견 3: 재정자립도는 유의하나 미미한 수준 (ρ = +0.192)**

232개 시군구 중 재정자립도와 입양률의 상관은 통계적으로 유의(p = 0.006)하지만, 실질적 효과 크기가 매우 작다. 재정이 열악한 시군구의 낮은 입양률은 재정 자체보다는 재정 부족에 따른 **보호소 인력/시설 미비, 입양 프로그램 부재** 등 간접 경로를 통한 것으로 추정된다.

### 6.2 정책적 함의

| 기존 접근 (비효과적) | 데이터 기반 접근 (권장) |
|---------------------|----------------------|
| 대도시에 입양센터 확대 | 성과 상위 보호소 운영 모델 확산 |
| 인구 많은 지역 우선 투자 | 시군구 단위 보호소 역량 진단 |
| 재정 지원만으로 입양률 제고 기대 | 재정 + 운영 프로그램 패키지 지원 |
| 대형 보호소 신설 | 적정 규모 분산형 네트워크 |

## 7. 가설 검증과의 연결

### 7.1 기존 가설 검증 결과 재확인

| 기존 가설 | 기존 판정 | 외부 데이터 검증 |
|----------|----------|----------------|
| H3: 광역시 > 도 | 기각 | **재확인** — 인구밀도 ρ ≈ 0으로 도시/농촌 무관 확정 |
| H5: 규모 → 입양률 | 기각 | **부분 보완** — 시군구 집계에서는 약한 양의 상관, 생태학적 오류 가능성 |
| H6: 도 내 격차 | 채택 | **강화** — 사회경제 변수로 격차의 3%만 설명, 나머지 97%는 운영 역량 |

### 7.2 새로운 인사이트

- **생태학적 오류 발견**: 보호소 단위(H5)에서 규모↑ → 입양률↓이었으나, 시군구 단위에서는 규모↑ → 입양률↑로 방향이 반전. 분석 단위에 따라 결론이 달라질 수 있으므로, 향후 분석에서 집계 수준을 명시적으로 통제해야 함.

- **"설명되지 않는 분산" 자체가 핵심 발견**: R² = 0.031이라는 결과는 실패가 아니라, "입양률은 사회경제 구조가 아닌 운영 역량이 결정한다"는 가장 강력한 증거다.

### 8.1 데이터 한계

- **서울 시군구 인구 데이터 미수집**: 26개 구의 인구밀도가 결측. 전체 매칭률 87.9%
- **고령화율/1인가구비율이 시도 단위**: 시군구 수준 편차를 반영하지 못함. 시군구 단위 데이터 확보 시 재분석 필요
- **면적 데이터 기준 연도(2020)와 분석 기준(2023) 불일치**: 실질적 영향은 미미하나 엄밀성에서 한계
- **횡단면 분석의 한계**: 시간 변화(재정자립도 개선 → 입양률 변화)를 추적할 수 없음

### 8.2 후속 분석 제안

| 우선순위 | 과제 | 기대 효과 |
|---------|------|----------|
| 1 | **보호소 클러스터링**: 입양률·안락사율·자연사율 조합으로 보호소 유형 분류 | "어떻게 운영하는가"의 정량적 유형화 |
| 2 | **개체 수준 분류 모델**: 나이·품종·중성화·지역 등으로 입양 확률 예측 | 개별 동물의 입양 가능성 스코어링 |
| 3 | **패널 데이터 분석**: 2019~2024 연도별 외부 데이터로 종단 분석 | 재정자립도 변화 → 입양률 변화의 동적 관계 |
| 4 | **보호소 운영 데이터 연계**: 직영/위탁, 인력 수, 프로그램 유무 | R² 향상의 핵심 후보 변수 |
