| 지표 | 2019 → 2024 | 변화 |
|------|-------------|------|

## 5. 기존 분석과의 교차 검증

### 5.1 API 분석 결과 vs 실태조사 비교

| 항목 | API 분석 (Homebound) | 실태조사 (정부 공식) | 일치 여부 |
|------|-------------------|----------------|---------|
| 위탁 센터 수 | 325개 (매칭된 보호소) | 156~188개 (위탁) | 수치 차이 있음 (집계 기준 상이) |
| 직영 추정 | 239개 (미매칭) | 75개 (직영) | API에서 직영 미제공 확인 |
| 센터유형 분류 | 법인/동물병원/단체/개인 | 직영/시설위탁/위탁보호 | 분류체계 상이 |
| 동물병원 위탁 입양률 최고 | 41.8% | (직접 비교 데이터 없음) | 실태조사에서도 위탁보호가 다수 |

**해석**: API는 위탁 보호센터만 포함하는 데이터였으므로, 매칭된 325개 중 실제 위탁은 156~188개 수준. 나머지는 한 시설이 여러 지자체에서 지정받은 중복 건 등으로 추정. API 미매칭 239개 = 직영 추정은 **과대 추정**이며, 실제 직영은 75개소에 불과.

### 5.2 "Where가 아닌 How" 가설의 실태조사 검증

**Homebound 프로젝트 핵심 가설**: 입양률은 지역(Where)이 아닌 운영 방식(How)에 의해 결정됨.

실태조사가 제공하는 추가 증거:

**(1) 투입 vs 성과의 비례 관계 부재**
- 비용 2배, 인력 22% 증가에도 입양률 2.9%p 하락
- "돈과 사람을 더 넣는다고 입양률이 오르지 않는다"

**(2) 서울의 역설**
- 서울: 직영 2개소, 인력 47명(2022), 비용 9.5억(2024), 보호기간 14.5일(최단)
- 경기: 직영 9개소, 인력 193명, 비용 126.6억, 보호기간 27일
- 서울은 자원 투입이 적지만 회전율이 높음 → 운영 효율성의 차이

**(3) 직영 확대의 한계**
- 직영 비율 18.7% → 32.5% 증가에도 입양률 하락
- 직영이라는 "형태" 자체가 성과를 보장하지 않음
- 이는 기존 분석에서 직영/위탁 입양률 차이가 무의미(p=0.761)했던 결과와 정확히 일치

**(4) 자연사 증가의 의미**
- 인도적 처리 21.8% → 18.5%로 감소(긍정적 외관)
- 그러나 자연사 24.8% → 27.5%로 증가
- 합산 사망률(자연사+인도적 처리): 46.6% → 46.0%로 거의 불변
- "안락사를 줄인 것이 아니라, 보호소에서 죽도록 방치하는 비율이 증가"

### 5.3 외부데이터 분석(R²=0.031) 결과의 보강

기존 분석에서 사회경제 변수(재정자립도, 인구밀도 등)가 입양률의 3.1%만 설명한다는 결과를 얻었다. 실태조사 데이터는 이를 다른 각도에서 보강:

- 서울(재정자립도 최상) vs 경북(재정자립도 하위): 경북 입양률(32.6%)이 서울보다 높음
- 경기(운영비용 126.6억, 전국 최고) vs 대구(10.0억): 대구 입양률(32.5%)이 경기보다 높음
- **부유한 지역, 자원이 많은 지역이 반드시 높은 입양률을 보이지 않음**

## 8. 스토리텔링 v2.0 보강 포인트

실태조사 데이터로 기존 스토리텔링의 근거를 강화할 수 있는 포인트:

### ACT 4 (검증) 보강
- **H1(계절성)**: 구조 건수 시기별 데이터 — 5월·7월 최고(10.1%), 12월 최저(6.6%) → 봄철 유기 급증 확인
- **H3(지역 사회경제)**: 실태조사 지역별 비용 vs 입양률 교차 분석 → "돈이 아니다" 재확인
- **H7(센터 운영역량)**: 인력/비용 2배 증가에도 입양률 하락 → "양이 아닌 질"

### ACT 5 (처방) 보강
- **정책1(광역 공동 보호체계)**: 직영 미설치 7개 지역 명시 → 정책 타겟 구체화
- **정책2(운영 모델 벤치마킹)**: 서울(보호기간 14.5일), 대구(입양률 32.5%) 등 벤치마크 사례 명시
- **정책3(봄철 TNR)**: TNR 사업 확대(2.5배)와 구조 건수 감소(-21%)의 상관관계 근거
- **정책5(농촌→도시 이송)**: 전남(보호기간 39.3일) vs 서울(14.5일) → 이송 효율성 근거
- **추가 정책**: 자연사율 증가(+2.7%p) 대응 — 보호 환경 질 개선 필요

## 9. 결론: "투입의 역설"

2019~2024 실태조사 6년치 데이터가 말하는 것은 명확하다.

**직영 센터 2배 증가, 운영비용 2배 증가, 인력 22% 증가에도 불구하고 입양률은 오히려 2.9%p 하락했다.** 자연사율은 2.7%p 증가하여, 인도적 처리 감소분을 상쇄했다.

이는 Homebound 프로젝트의 핵심 메시지 **"Where가 아닌 How"**를 가장 강력하게 뒷받침하는 거시적 증거다. 어디에 센터를 짓고, 얼마를 투입하느냐가 아니라, **어떻게 운영하느냐**가 입양률을 결정한다.

구체적으로:
- 동물병원 위탁(전문성)이 개인 위탁보다 18.5%p 높은 입양률 (API 분석)
