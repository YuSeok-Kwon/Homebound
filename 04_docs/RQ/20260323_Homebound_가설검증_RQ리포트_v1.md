## 1. 핵심 개념 가이드

분석에 사용된 통계 기법을 쉽게 풀어서 설명합니다.

### (1) 카이제곱 검정 (Chi-square Test)

두 범주형 변수 사이에 관련이 있는지 확인하는 방법입니다.

```
예시: "나이그룹"과 "입양 여부" 사이에 관련이 있는가?
→ 관련이 없다면(독립이라면), 모든 나이그룹의 입양률이 비슷해야 합니다.
→ 실제로 큰 차이가 있다면, "나이가 입양에 영향을 미친다"고 판단합니다.
```

> **p-value < 0.05**: 우연히 이런 차이가 발생할 확률이 5% 미만 → "통계적으로 유의하다"

### (2) Mann-Whitney U 검정

두 그룹의 분포가 다른지 비교하는 비모수 검정입니다. 데이터가 정규분포를 따르지 않아도 사용할 수 있습니다.

```
예시: 광역시 7곳의 입양률 vs 도 10곳의 입양률
→ 두 그룹의 입양률 분포가 통계적으로 다른가?
```

### (3) 스피어만 상관계수 (Spearman's rho)

두 변수 사이의 단조 관계(한쪽이 크면 다른 쪽도 크거나 작은 관계)를 측정합니다.

| rho 값 | 해석 |
|--------|------|
| 0.7 ~ 1.0 | 강한 양의 상관 |
| 0.3 ~ 0.7 | 중간 양의 상관 |
| 0 ~ 0.3 | 약한 양의 상관 |
| -0.3 ~ 0 | 약한 음의 상관 |
| -1.0 ~ -0.3 | 중간~강한 음의 상관 |

### (4) 변동계수 (CV, Coefficient of Variation)

평균 대비 데이터가 얼마나 퍼져 있는지를 나타냅니다. CV가 클수록 격차가 심하다는 뜻입니다.

```
CV = 표준편차 / 평균
CV > 0.3이면 "상당한 격차가 존재한다"고 해석합니다.
```

## 2. 왜 이 분석을 하는가?

### 비즈니스 배경

전국 보호종료동물 789,874건(2019~2025) 중 입양률은 31.2%에 불과합니다. **52.1%의 동물이 보호소에서 사망**(자연사 29.3% + 안락사 22.8%)하고 있으며, 정부의 제3차 동물복지 종합계획(2025~2029)은 유기동물 발생을 11.3만에서 6만으로 감축하겠다는 목표를 제시했습니다.

- **현상**: 입양률 31.2%, 사망률 52.1%로 보호소가 "생명의 종착역"이 되고 있음
- **질문**: 어떤 요인이 입양 성공에 영향을 미치며, 어디에 정책 자원을 집중해야 하는가?
- **기대 효과**: 데이터 기반 정책 제언으로 입양률 향상 방안 도출

### 연구 질문 (3개 축)

| 축 | 핵심 질문 | 관련 가설 |
|----|----------|----------|
| 동물 특성 | 어떤 동물이 입양되고, 어떤 동물이 사각지대에 놓이는가? | H1, H2 |
| 지역/보호소 | 같은 동물이라도 보호소·지역에 따라 입양률이 다른 원인은? | H3, H5, H6 |
| 시간 패턴 | 유기동물 발생과 입양에 시간적 패턴이 있으며, 선제 대응이 가능한가? | H4, H7 |

## 5. 가설 검증 결과

### 가설 1: 어린 동물일수록 입양률이 높다

**가설 설정**

```
H0: 나이그룹과 입양 여부는 독립이다 (나이가 입양에 영향을 미치지 않는다)
H1: 나이그룹에 따라 입양률이 유의하게 다르다
```

> **왜 이 가설을 세웠나?**
> 어린 동물은 사회화 가능성이 높고 입양자의 선호도가 높다는 것이 일반적인 인식이다. 이를 데이터로 검증하여, 고령 동물의 입양 사각지대를 정량화하고자 했다.

**검증 결과**

| 나이그룹 | 총 건수 | 입양률 |
|----------|---------|--------|
| 60일미만 | 110,045 | 34.3% |
| 1세 이하 | 374,466 | 33.7% |
| 2-3세 | 159,069 | 27.6% |
| 4-5세 | 68,149 | 25.0% |
| 6-7세 | 25,826 | 22.9% |
| 8-10세 | 21,508 | 17.8% |
| **11세 이상** | **6,069** | **10.2%** |

```
검정 통계량: χ² = 7,673.2
자유도: df = 6
p-value: < 0.001
```

![나이그룹별 입양률](../../02_outputs/figures/가설1_나이별입양률.png)

**결과 해석**

- **판정: ✅ 채택** — 나이그룹 간 입양률 차이는 통계적으로 극히 유의 (p < 0.001)
- 나이가 많아질수록 입양률이 **단조 감소**하는 뚜렷한 패턴 확인
- 1세 이하(33.7%)가 40% 기준에는 미달하나, 11세 이상(10.2%)은 15% 미만 기준 충족
- 8세 이상 동물 27,577건 중 입양은 약 4,500건에 불과 → **고령 동물 입양 사각지대** 확인
- 개와 고양이 모두 동일한 단조 감소 패턴을 보이며, 고양이의 전 연령대 입양률이 개보다 약간 높음

### 가설 2: 중성화된 동물의 입양률이 유의미하게 높다

**가설 설정**

```
H0: 중성화 여부와 입양 여부는 독립이다
H1: 중성화된 동물의 입양률이 비중성화 동물보다 10%p 이상 높다
```

> **왜 이 가설을 세웠나?**
> 경기도 용인시(입양률 82%)는 중성화/백신접종 무료 제공을 입양률 향상의 핵심 전략으로 삼고 있다. 중성화가 입양 촉진에 미치는 효과를 전국 데이터로 검증하고자 했다.

**검증 결과**

| 구분 | 중성화 O | 중성화 X | 차이 |
|------|---------|---------|------|
| **전체** | **32.9%** | **32.1%** | **+0.7%p** |
| 개 | 32.5% | 30.9% | +1.6%p |
| 고양이 | 34.4% | 35.1% | -0.7%p |

```
카이제곱: χ² = 8.5, p = 3.61e-03
비율 Z-검정: Z = 2.91
```

![중성화별 입양률](../../02_outputs/figures/가설2_중성화별입양률.png)

**결과 해석**

- **판정: ⚠️ 수정채택** — 통계적으로 유의하나(p < 0.01), 실질적 차이는 0.7%p로 예상(10%p)에 크게 미달
- **통계적 유의성 vs 실질적 유의성**: p-value는 유의하지만 이는 대규모 데이터(수십만 건)에서 아주 작은 차이도 유의하게 나타나는 현상. **실무적으로 0.7%p 차이는 무의미에 가까움**
- 고양이는 오히려 중성화 X가 미세하게 높음 → 중성화 자체가 입양을 결정하는 요인은 아닌 것으로 판단
- **주의**: 이 결과는 집계 데이터의 한계일 가능성이 높음. 중성화 시점(입소 전/후)과 보호소별 중성화 정책이 통제되지 않았으며, 용인시처럼 입양 전 무료 중성화를 제공하는 보호소의 효과는 이 데이터로 분리하기 어려움

### 가설 3: 광역시의 입양률이 도 지역보다 높다

**가설 설정**

```
H0: 광역시와 도의 입양률 분포는 동일하다
H1: 광역시의 입양률이 도보다 유의하게 높다 (5~10%p 차이)
```

> **왜 이 가설을 세웠나?**
> 도시 지역은 인구 밀도가 높아 잠재 입양자 풀이 크고, 보호소 접근성이 높을 것으로 예상했다. 고창군/완도군 사례에서 보듯 농촌 지역은 입양 수요 자체가 부족하다는 관찰에서 출발했다.

**검증 결과**

| 유형 | 시도 수 | 가중평균 입양률 | 주요 지역 |
|------|--------|----------------|----------|
| 광역시 | 7 | **29.4%** | 대구 35.8%, 인천 34.8%, 울산 22.0% |
| 도 | 10 | **31.2%** | 전북 43.3%, 경북 40.9%, 제주 14.9% |

```
Mann-Whitney U = 25, p = 0.3638
차이: -1.8%p (도가 오히려 높음)
```

![광역시 vs 도](../../02_outputs/figures/가설3_광역시vs도.png)

**결과 해석**

- **판정: ❌ 기각** — 통계적으로 유의하지 않으며(p = 0.36), 방향도 예상과 반대
- "도시 = 입양률 높음"이라는 상식이 데이터로 **깨졌음**
- 전북(43.3%), 경북(40.9%) 등 **도 지역 일부가 광역시보다 훨씬 높은 입양률**을 보임
- 반면 울산(22.0%), 광주(23.5%), 부산(24.4%) 등 광역시 중에도 낮은 곳이 존재
- **핵심 인사이트**: 입양률은 도시/농촌이라는 이분법보다 **개별 보호소의 운영 역량과 지역 정책**에 더 크게 좌우됨. 광역시-도 프레임이 아닌, **시군구-보호소 단위**의 분석이 필요

### 가설 4: 여름철(6~8월) 유기동물 발생 증가, 특히 고양이

**가설 설정**

```
H0: 월별 유기동물 발생 건수는 균등하다
H1: 여름(6~8월) 발생이 월평균 대비 유의하게 높으며, 고양이에서 특히 뚜렷하다
```

> **왜 이 가설을 세웠나?**
> 고양이는 봄(3~4월) 번식기에 태어난 새끼가 여름에 유기되는 생물학적 패턴이 존재한다. 이를 데이터로 확인하면 봄철 사전 중성화 사업의 근거가 된다.

**검증 결과**

| 동물 | 월평균 대비 여름(6~8월) | 카이제곱 | p-value |
|------|----------------------|---------|---------|
| **고양이** | **+48.5%** | 47,085.3 | < 0.001 |
| 개 | +5.0% | 1,865.7 | < 0.001 |

![여름철 발생 패턴](../../02_outputs/figures/가설4_여름철발생.png)

**결과 해석**

- **판정: ✅ 채택** — 고양이 여름 발생이 월평균 대비 48.5% 높아 예상 기준(30%) 초과 달성
- 개는 월별 변동이 상대적으로 작고(+5.0%), 여름 집중 현상이 미미
- **고양이는 5~8월 4개월간 연간 발생의 약 45%가 집중** → 봄 번식기 패턴과 정확히 일치
- 개도 균등분포는 기각되지만(p < 0.001), 여름 집중도는 고양이의 1/10 수준
- **정책 함의**: 봄철(3~5월) 길고양이 TNR 사업을 집중적으로 시행하면 여름 유입량을 효과적으로 감소시킬 수 있음

### 가설 5: 보호소 규모가 클수록 입양률이 높다

**가설 설정**

```
H0: 보호소 규모(총 보호건수)와 입양률 사이에 상관관계가 없다
H1: 양의 상관관계가 존재한다 (ρ > 0.3)
```

> **왜 이 가설을 세웠나?**
> 규모가 큰 보호소는 전담 인력, 입양 프로그램, 의료 지원 등 인프라가 갖춰질 가능성이 높다. 고창군(50마리, 4명) vs 군산시(500마리, 13명) 비교에서 규모와 입양률의 양의 상관이 시사되었다.

**검증 결과**

| 보호소 규모 | 보호소 수 | 평균 입양률 | 평균 안락사율 |
|------------|----------|-----------|-------------|
| ~500건 | 83 | 41.1% | 22.3% |
| 500~1K | 75 | 35.6% | 21.9% |
| 1K~3K | 156 | 30.4% | 21.3% |
| 3K~10K | 69 | 31.4% | 21.0% |
| **10K~** | **5** | **30.0%** | **30.1%** |

```
Spearman ρ = -0.165, p = 0.0011
```

![보호소 규모 vs 입양률](../../02_outputs/figures/가설5_보호소규모.png)

**결과 해석**

- **판정: ❌ 기각 (방향 반대)** — 상관계수가 음수(ρ = -0.165). 규모가 클수록 입양률이 오히려 **낮아지는** 패턴
- 소규모 보호소(~500건)의 평균 입양률 41.1%는 대규모(10K~)의 30.0%보다 11.1%p 높음
- 초대형 보호소(10K~)는 안락사율도 30.1%로 급등 → **수용 과부하에 따른 개별 관리 한계**
- **핵심 인사이트**: "크면 좋다"가 아니라 **"적정 규모 + 운영 품질"이 핵심**. 대형 집중형보다 소규모 분산형 + 광역 네트워크 모델이 더 효과적일 수 있음
- 다만 이 결과는 역인과(입양률이 높은 곳은 동물이 빨리 빠져나가 보호건수가 적은 것일 수 있음)의 가능성이 있으므로 해석에 주의 필요

### 가설 6: 동일 도 내 시군구별 입양률 격차가 크다

**가설 설정**

```
H0: 동일 도 내 시군구의 입양률 격차가 작다 (CV < 0.3, 격차 < 30%p)
H1: 격차가 크다 (CV > 0.3, 최고-최저 격차 > 30%p)
```

> **왜 이 가설을 세웠나?**
> 현행 시군 단위 개별 운영 구조에서는 인접 지역이라도 보호소 역량에 따라 성과가 크게 다를 수 있다. 이를 정량화하여 광역 공동 보호체계의 필요성 근거를 제시하고자 했다.

**검증 결과**

| 도 | 시군구 수 | CV | 최고-최저 격차 | 최고 지역 | 최저 지역 |
|----|---------|-----|-------------|----------|----------|
| 경북 | 22 | 0.465 | **61.1%p** | 칠곡군(72.1%) | 청송군(11.0%) |
| 경기 | 31 | 0.385 | 52.7%p | 가평군(67.3%) | 연천군(14.6%) |
| 강원 | 18 | 0.399 | 48.8%p | 평창군(61.7%) | 횡성군(12.9%) |
| 충남 | 15 | 0.358 | 48.0%p | 아산시(59.5%) | 서천군(11.5%) |
| 전북 | 14 | 0.294 | 45.3%p | 김제시(55.9%) | 고창군(10.6%) |
| 전남 | 22 | 0.472 | 44.5%p | 여수시(50.2%) | 완도군(5.7%) |
| 충북 | 11 | 0.467 | 37.0%p | 충주시(47.6%) | 영동군(10.7%) |
| 경남 | 18 | 0.427 | 36.8%p | 고성군(48.1%) | 창녕군(11.3%) |

```
전체 평균 CV: 0.408 (기준 0.3 초과)
전체 평균 격차: 46.8%p (기준 30%p 초과)
CV > 0.3인 도: 7/8개 (87.5%)
격차 > 30%p인 도: 8/8개 (100%)
```

![도 내 시군구 격차](../../02_outputs/figures/가설6_도내격차.png)

**결과 해석**

- **판정: ✅ 채택** — 모든 도에서 30%p 이상의 시군구 간 입양률 격차가 존재
- 경북도 내에서 칠곡군(72.1%) vs 청송군(11.0%)은 **61.1%p 차이** — 같은 도 내에서 6배 이상의 성과 차이
- **동일한 법적 프레임워크, 동일한 도비 지원** 아래에서도 이만큼의 격차 → 개별 보호소의 운영 역량이 결정적
- 이 결과는 **광역 공동 보호체계**의 필요성을 가장 강하게 뒷받침하는 근거임
- 최저 지역들(청송 11.0%, 고창 10.6%, 완도 5.7%, 영동 10.7%)은 공통적으로 **군 단위 농어촌** → 인구 과소, 보호소 인력/시설 부족이라는 구조적 원인

### 가설 7: 연도별 발생 감소 추세, 정책 변곡점 존재

**가설 설정**

```
H0: 연도별 발생 건수에 추세가 없다 (기울기 = 0)
H1: 통계적으로 유의한 감소 추세가 존재한다
```

> **왜 이 가설을 세웠나?**
> 2019년 이후 발생 건수가 지속 감소하는 것으로 보이지만, 이것이 우연인지 의미 있는 추세인지 확인하고, 정책(동물보호법 개정 등)과의 관계를 분석하고자 했다.

**검증 결과**

| 연도 | 발생 건수 | 전년비 |
|------|----------|--------|
| 2019 | 127,519 | - |
| 2020 | 123,264 | -3.3% |
| 2021 | 113,179 | -8.2% |
| 2022 | 108,938 | -3.7% |
| 2023 | 108,317 | -0.6% |
| 2024 | 100,793 | -6.9% |

```
선형회귀: 기울기 = -5,220건/년, R² = 0.952, p = 0.0009
전반부(2019~2021): -7,170건/년
후반부(2022~2024): -4,072건/년
→ 후반부 감소 속도 둔화
```

![연도별 추세](../../02_outputs/figures/가설7_연도별추세.png)

**결과 해석**

- **판정: ✅ 채택** — 6년간 21.0% 감소, R² = 0.952로 매우 강한 선형 감소 추세
- 전반부(2019~2021) 감소 속도(-7,170건/년)가 후반부(2022~2024, -4,072건/년)보다 빨라 **감소세 둔화** 관찰
- 2023년 감소율 -0.6%로 거의 정체 → 추가 정책 없이는 감소 한계에 도달할 가능성
- 개는 꾸준히 감소하나, 고양이는 감소 폭이 작아 **향후 고양이 비중이 더 높아질 것**으로 예상
- 정책과의 연결: 2022년 동물보호법 개정 시행 이후에도 감소 추세가 유지되고 있으나, 감소 속도가 둔화된 것은 **쉽게 줄일 수 있는 부분은 이미 줄었고**, 구조적 문제(길고양이, 농촌 마당개 등)가 남아 있음을 시사

## 6. 종합 판정 및 해석

### 6.1 가설별 판정 요약

| 가설 | 판정 | p-value | 핵심 수치 |
|------|------|---------|----------|
| H1: 어린 동물 → 입양률↑ | **✅ 채택** | < 0.001 | 1세이하 33.7% → 11세+ 10.2% |
| H2: 중성화 → 입양률↑ | **⚠️ 수정채택** | 0.004 | 차이 0.7%p (유의하나 실질적 미미) |
| H3: 광역시 > 도 | **❌ 기각** | 0.364 | 오히려 도(31.2%) > 광역시(29.4%) |
| H4: 여름 고양이 발생↑ | **✅ 채택** | < 0.001 | 고양이 +48.5%, 개 +5.0% |
| H5: 규모↑ → 입양률↑ | **❌ 기각** | 0.001 | **방향 반대**: ρ = -0.165 |
| H6: 도 내 격차 극심 | **✅ 채택** | - | 평균 CV 0.408, 격차 46.8%p |
| H7: 연도별 감소 | **✅ 채택** | 0.001 | -5,220건/년, R² = 0.952 |

**채택/수정채택: 5/7 (71.4%) | 기각: 2/7 (28.6%)**

### 6.2 기각된 가설이 주는 인사이트

기각된 가설이 오히려 더 중요한 인사이트를 제공했습니다.

**H3 기각 — "도시 = 높은 입양률"은 편견이었다**

입양률을 결정하는 것은 도시/농촌 구분이 아니라 **개별 보호소의 운영 역량과 지역 정책**입니다. 전북(43.3%), 경북(40.9%)처럼 도 단위에서도 높은 입양률을 달성한 곳이 있고, 울산(22.0%), 광주(23.5%)처럼 광역시에서도 낮은 곳이 있습니다. 이는 단순히 "도시에 입양센터를 더 짓자"가 아니라, **성과가 높은 보호소의 운영 모델을 분석하고 확산**하는 것이 더 효과적임을 시사합니다.

**H5 기각 — "크면 좋다"는 오산이었다**

보호소 규모가 클수록 입양률이 오히려 낮아지는 역상관(ρ = -0.165)은 **대형 집중형 보호소의 한계**를 보여줍니다. 소규모 보호소(~500건)의 평균 입양률 41.1%는 초대형 보호소(10K~)의 30.0%보다 11%p 높습니다. 이는 정부의 "직영 보호소 76 → 130개소 확대" 계획에서 **양적 확대보다 적정 규모 + 운영 품질**에 집중해야 함을 시사합니다.

## 7. 결론 및 실행 제언

### 7.1 핵심 인사이트 (5대 발견)

1. **나이가 입양의 가장 강력한 예측 변수** — 1세 이하(33.7%) → 11세 이상(10.2%)으로 단조 감소. 고령 동물 27,577건 중 대부분이 보호소에서 생을 마감함

2. **"도시 vs 농촌" 프레임은 틀렸다** — 입양률은 광역시/도 구분보다 개별 보호소·지자체의 운영 역량에 좌우됨. 전북(43.3%)이 서울(30.9%)보다 높음

3. **같은 도 안에서 입양률 6배 차이** — 경북 칠곡(72.1%) vs 청송(11.0%). 8개 도 모두 30%p 이상 격차 → 광역 공동 보호체계의 필요성

4. **고양이는 여름에 집중, 개는 연중 균등** — 고양이 6~8월 발생이 월평균 대비 48.5% 높음. 봄철 TNR 사업의 타이밍 근거

5. **"큰 보호소 = 좋은 보호소"가 아님** — 규모↑ → 입양률↓ (ρ = -0.165). 적정 규모 분산형 + 네트워크 모델이 더 효과적

### 7.2 비즈니스 실행 제언 (Action Plan)

| 순위 | 실행 과제 | 우선순위 | 근거 | 관련 가설 |
|------|-----------|---------|------|----------|
| 1 | **광역 공동 보호체계 구축**: 도 단위 거점 보호소 + 군 임시보호 네트워크 | 높음 | 8개 도 모두 시군구 격차 30%p↑, 소규모 보호소 입양률 우위 | H5, H6 |
| 2 | **봄철(3~5월) 고양이 TNR 집중 시행**: 여름 유입 사전 차단 | 높음 | 고양이 여름 발생 +48.5%, 개의 10배 집중도 | H4 |
| 3 | **고령·대형견 입양 촉진 프로그램**: 입양비 추가 지원, 의료비 보조 | 중간 | 8세+ 입양률 10~18%, 전체 평균의 1/3 수준 | H1 |
| 4 | **상위 보호소 운영 모델 벤치마킹**: 전북·경북 사례 분석 및 확산 | 중간 | 도 단위에서도 40%↑ 달성 사례 존재, 광역시보다 높은 성과 | H3 |
| 5 | **감소세 둔화 대응**: 2023~2024년 정체 구간 원인 분석 및 추가 정책 | 중간 | 감소 기울기 후반부 둔화(-7,170 → -4,072건/년) | H7 |

### 7.3 한계점 및 후속 분석

**데이터의 한계**

- **보호소 운영 주체 데이터 없음**: 직영/위탁/민간 구분이 없어 운영 모델별 성과 비교 불가
- **보호소 인력·예산 데이터 없음**: 총보호건수를 규모 대리변수로 사용했으나, 실제 인력당 보호동물 수 등 운영 효율 지표 산출 불가
- **집계 데이터의 한계**: H2(중성화)에서 중성화 시점, 보호소별 정책 등이 통제되지 않아 효과 크기가 희석되었을 가능성
- **입양자 정보 없음**: 입양 동기, 입양자 인구통계 등 수요측 분석 불가

**후속 분석 제안**

1. **개체 수준 분류 모델**: 입양 확률에 영향을 미치는 변수를 종합적으로 모델링 (로지스틱 회귀, Random Forest 등)
2. **보호소 클러스터링**: 입양률·안락사율·자연사율 조합으로 보호소 유형을 분류하고, 각 유형별 개선 전략 도출
3. **시계열 예측**: 지역별 유기동물 발생 추세를 예측하여 사전 자원 배분 최적화
4. **외부 데이터 연계**: 인구통계(인구밀도, 1인가구 비율), 재정자립도 등과 입양률의 관계 분석
